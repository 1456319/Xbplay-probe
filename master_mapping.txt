Xbox ONE Game Streaming App - Master File Functionality Map
===========================================================

This document provides a comprehensive classification of the individual files and components within the application to detail their likely purpose. Open-source libraries flagged as copyleft violations have been excluded from this mapping.

### Core Application Packages
- `base.apk`: The primary application package containing compiled code, core resources, and assets.
- `split_config.arm64_v8a.apk`, `split_config.en.apk`, `split_config.xxhdpi.apk`: Architecture-specific and resource-specific application splits.

### Web Assets and Bridging (`base.apk/assets/`)
- `base.apk/assets/old.html`: Front-end web asset containing Microsoft Xbox Cloud Gaming API endpoints (`gssv-play-prodxhome.xboxlive.com`) and client logic (`xSDK client.js`). Functions include establishing WebRTC peer connections, handling ICE candidate negotiation, creating data channels (video, audio, input, control, message, chat), and managing media source rendering.
- `base.apk/assets/messaging/background.js`: Content script acting as a messaging bridge between the Android host and the GeckoView web view. Employs a recursive `isValidData` function to ensure only safe, plain objects and primitives are transferred via nativeMessaging and CustomEvents, preventing unsafe data cloning (such as functions or prototypes).
- `base.apk/res/raw/omsdk_v_1_0.js`: Contains the IAB Open Measurement SDK (v1.3.30-iab3294), functioning as a tool for ad viewability measurement.

### Authentication (`com/studio08/xbgamestream/Authenticate`)
- `LoginActivity.java`, `LoginClient.java`: Manage Xbox Live login interfaces and the retrieval/renewal of authentication tokens necessary for SmartGlass connectivity.

### Web View and Tokens (`com/studio08/xbgamestream/Web`)
- `StreamWebview.java`, `ApiClient.java`, `SmartglassClient.java`: Responsible for rendering Microsoft OAuth login pages and interfacing with Xbox web APIs.

### Controller Input Handling (`com/studio08/xbgamestream/Controller`)
- `ControllerHandler.java`, `RumbleHelper.java`, `InputDeviceContext.java`: Interface with Android's `InputDevice` APIs. They read physical gamepad inputs, apply rumble feedback, and package the inputs into Nano `ControllerPacket` structures.

### Controller Configuration (`com/studio08/xbgamestream/ControllerSetup`)
- Components handling the configuration and mapping setup for connected physical and virtual input devices.

### Media Servers (Casting) (`com/studio08/xbgamestream/Servers`)
- `FileServer.java`, `Xbox360FileServer.java`, `TvServer.java`: Built-in local HTTP or UPnP/DLNA servers designed to host local files for casting to the Xbox Media Player.

### License/Purchase Validation (`com/studio08/xbgamestream/Timers`)
- `PurchaseChecker.java`, `PCheckInterface.java`: Implement logic for verifying application purchases, licenses, or premium status unlock mechanisms.

### Home Screen Widgets (`com/studio08/xbgamestream/Widgets`)
- `PowerWidgetProvider.java`, `RemoteWidgetProvider.java`: Android Home Screen widgets enabling basic console power management and media remote commands without requiring the full application interface.

### User Interface Fragments (`com/studio08/xbgamestream/ui`)
- `gamestream/GamestreamFragment.java`, `controller/ControllerFragment.java`, `remote/RemoteFragment.java`: Visual components (fragments) presenting views for streaming video, touch controls, and media casting interfaces.

### Video Format Conversion (`com/studio08/xbgamestream/Converter`)
- `VideoFormatConverter.java`, `ConvertForegroundService.java`: Components that manage the transcoding or conversion of local media files into formats compatible with the Xbox Media Player prior to casting.

### Media Casting (`com/studio08/xbgamestream/Cast` & `CastToConsole`)
- `CastOptionsProvider.java`, `MyExpandedControls.java`: Integrations for media casting protocols.
- `CastToConsoleForegroundService.java`: A background service managing active media casting sessions to the Xbox console.

### SmartGlass Cryptography (`crypto`)
- `SgCrypto.java`: Executes Elliptic Curve Diffie-Hellman (ECDH) key exchange and AES-CBC encryption/decryption, satisfying the security requirements for SmartGlass protocol communication.

### Network Services and Protocol Bridging (`network`)
- `BindService.java`: A background Android Service bridging the application UI with the SmartGlass (`Xbox`) and Nano (`Nano`) clients. It manages connection states, controller sequences, and socket persistence.
- `SmartGlass.java`, `Listeners.java`, `EventListener.java`: Define network configurations and event listener interfaces.

### Nano Gamestream Core (`nano/base`)
- `Nano.java`: Core Nano gamestream client managing UDP and TCP sockets, processing initial handshakes, and routing gamestream packets.
- `VideoDecoder.java`: Utilizes Android's `MediaCodec` to decode the incoming H.264 video stream and render it to a `TextureView`.
- `RTPHeader.java`, `StreamerHeader.java`, `ControlHeader.java`, `h264Frame.java`: Parsers designed to reconstruct video frames and read stream headers.

### Nano Gamestream Channels (`nano/streams`)
- `CoreStream.java`, `VideoStream.java`, `ControlStream.java`, `ControllerInputStream.java`: Handlers for multiplexed streams within the Gamestream session, separating control packets from video and input data.

### Nano Gamestream Packets (`nano/packets`)
- Packet definition classes (e.g., `InputFrame`, `VideoClientHandshake`) for serializing handshakes, control messages, and controller input frames.

### Gamepad UI Module (`studio09/gameconrtollerforxbox`)
- `Gamepad.java`: Standalone activity supplying a digital gamepad interface, which interacts with `BindService` to transmit SmartGlass commands.

### Callback Interfaces (`Interfaces`)
- `SmartglassEvents.java`, `NanoStreamEvents.java`, `XboxEvents.java`, `ChannelEvents.java`: Define the event signatures required to asynchronously communicate state changes between protocol implementations, services, and the UI.

### Utility Functions (`util`)
- `Util.java`: Supplies utility functions for byte array manipulation, UUID processing, endianness conversions, and hexadecimal formatting.

### SmartGlass Channels (`channels`)
- `SystemInputChannel.java`, `TextInputChannel.java`, `SystemBroadcastChannel.java`, `ChannelManager.java`: Responsible for processing specific subsets of SmartGlass communications, such as text entry and media remote controls.

### SmartGlass Client Core (`xbox`)
- `Xbox.java`: Main SmartGlass client executing console discovery, connection initiation, channel management, system input transmission, and power command handling.

### SmartGlass Packets (`packet`)
- `MessagePacket.java`, `SimplePacket.java`: Classes dedicated to structuring, serializing, and deserializing the binary packets utilized in the SmartGlass protocol.

### Protocol Constants and Mappings (`constants`)
- `PacketTypes.java`, `SystemInputButtonMappings.java`, `GameControllerButtonMappings.java`, `VideoFormat.java`, `ChannelTypes.java`: House fixed byte sequences, flags, and mappings that translate logical inputs into binary formats required by the console.

### Application Root Activities (`app/src/main/java/com/studio08/xbgamestream/`)
- `MainActivity.java`: A placeholder activity for the rebuilt project that attempts to load native JNI libraries (such as `gkcodecs`).

### Web Assets and Bridging Tests (`base.apk/assets/`)
- `base.apk/assets/messaging/background.test.js`: Standalone test suite utilizing Node.js's native test runner to verify the behavior of the `background.js` messaging bridge.
- `base.apk/assets/messaging/background.js`: Content script acting as a messaging bridge between the Android host and the GeckoView web view. It securely transfers JSON payloads and CustomEvents.


### Other Web Assets (`extracted_assets/`)
- `play-anywhere.html`: A web asset containing a placeholder `STREAM_VIEW_URL`.
